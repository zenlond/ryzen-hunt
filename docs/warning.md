# WARNING: Compatibility Issues

This app uses **HTTP** instead of HTTPS. Modern Android versions block this for security, so follow the guide below to make it work.

## Android Support

* **Android 6 and below:** ✅ Works perfectly.
* **Android 7 & 8.1:** ⚠️ Not tested. If it doesn't work, follow the Android 9 instructions.
* **Android 9:** 🛠 Manual setup required. You must install a CA certificate.
* **Android 10 and above:** ❌ Does not work.

---

## How to setup

1.  **Install the APK:** Use the **patched APK** posted in my Telegram channel.
2.  **Run the Generator:** Start the generator and enter your **IPv4 address**.
3. **Transfer Files:** Move the generated `server.crt` and `server.key` files to your phone. Do not delete these files from the server folder, as they are necessary for the server to start.
4.  **Install Certificate:** * Go to your phone **Settings**.
    * Find **Security** -> **Encryption & Credentials**.
    * Select **Install a certificate** -> **CA certificate**.
    * Install the file generated by the tool.
5.  **Start the Server:** Once the certificate is installed, run the server and everything should work.
